# Программный модуль «Анализатор воронки найма» (Recruitment Funnel Analyzer)

Выпускная квалификационная работа (ВКР) студента группы ПИ-21 Комарова М.Н.
Направление подготовки: 09.03.03 «Прикладная информатика»
Кафедра: Прикладной информатики, ПГУТИ, 2026 г.

## Описание проекта
Данный программный модуль предназначен для автоматизации сбора, обработки и визуализации ключевых HR-метрик (конверсии этапов воронки найма, контроль задержек по SLA, расчет Time-to-fill). Система собирает транзакционные логи переходов соискателей из внешней СУБД (MySQL), проводит очистку данных (ETL) и строит интерактивные визуальные дашборды в реальном времени.

## Технологический стек
* **Frontend:** React, TypeScript, Tailwind CSS, SVG-графика
* **Backend:** Node.js (TypeScript / Express)
* **База данных (в архитектуре):** MySQL

## Архитектура решения
Проект реализован в рамках классического трехзвенного клиент-серверного подхода:
1. **Presentation Layer (Клиентская часть):** React SPA для визуализации дашбордов.
2. **Business Logic Layer (Серверная часть):** Node.js API для извлечения сырых логов и расчета математической модели HR-метрик.
3. **Data Layer (Уровень данных):** Реляционная база данных для хранения истории откликов (таблица `Application_History` в третьей нормальной форме).
